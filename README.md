# BOOMBOX Chatbot

A Generative AI chatbot built using:

- Python
- Streamlit
- Google Gemini
- LangChain
- Speech Recognition
- pyttsx3

## Features

- Voice Input
- AI Responses
- Text-to-Speech
- Chat History

## Installation

pip install -r requirements.txt

## Run

streamlit run main.py
